gosqlite3 by kuroneko

Go Interface for SQLite3.

A Go interface for SQLite3, modelled closely after the C interface. Very incomplete, and moving at a leisurely pace.

Dependencies

sqlite3

Install

Unpack go. patch go tree using the cgo.diff patch. Build go. Build gosqlite3. Enjoy

License
